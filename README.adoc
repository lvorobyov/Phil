Постнеклассические модели роста научного знания.
================================================
Реферат по курсу "История и философия науки"

== Содержание каталога

- link:intro.adoc[Введение].
- link:1.adoc[1-й], link:2.adoc[2-й] и link:3.adoc[3-й] разделы.
- link:concl.adoc[Заключение].
- link:glossary.adoc[Словарь терминов].
- link:links.txt[Перечень цитируемых источников].

В тексте встречаются ссылки вида `cite:[id]`, а в link:links.txt[перечне ссылок] есть запись, начинающаяся с идентификатора `id` латиницей.

== Создание печатной версии

Для создания печатной версии нужно выполнить в рабочем каталоге следующие команды:

```
make all
```

== Используемое ПО

Для выполнения вышеприведённых команд на ПК должны быть установлены GNU Make версии не ниже 4.3, один из дистрибутивов LaTeX, BibTeX и Perl.
